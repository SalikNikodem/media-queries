# media-queries
Simple html site using media queries for dynamic change of layout depending on site width

Instalation
```
git clone https://github.com/SalikNikodem/media-queries.git
```
Basically, the html site represents different layout of div rectangles which changes it's position depending on user device.

Widths:
	> 1000px
	1000px > width > 600px
	< 600px
